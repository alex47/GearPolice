GearPolice TODO
===============

Medium priority
---------------

- Make target-scan retry behavior distinct from group-scan retry behavior.
  /gearpolice target can scan a non-group target, but RetryInspection currently
  marks non-group GUIDs as failed when inspection cannot start immediately.

Cleanup
-------

- Avoid direct AceGUI widget internals where practical.
  UI.lua mutates widget internals such as playerNameLabel.label and itemIcon.frame
  backdrop state. AceGUI documentation warns that direct frame mutations can leak
  through pooled widgets.

- Remove the ignored extra argument from CancelTimer calls.
  AceTimer:CancelTimer accepts only the timer id. The current true argument is
  ignored by the bundled AceTimer and should be removed for clarity.

- Remove or consolidate the duplicate HeadSlot entry in Inspection.lua.

- Decide whether Shared.lua should contain shared code or be removed from the
  load list.
